## Overview

### The challenge

Users should be able to:

- Generate a password based on the selected inclusion options
- Copy the generated password to the computer's clipboard
- See a strength rating for their generated password
- View the optimal layout for the interface depending on their device's screen size
- See hover and focus states for all interactive elements on the page

### What I learned

I built this primarily to practice using `useReducer` in react, and i anticipated that the challenge will be hard but it was not. I learnt several things in this challenge, such as:

- using `navigator.clipboard.writeText(text);` to copy texts to the clipboard

### Extra Features

I also added some extra features to this project such as:

- The user should be able to view the history of previously generated passwords
- the user will be able to copy previously generated passwords from the history modal
- the user will be able to delete any particular password from history
- the user will be able to delete all the passwords from history
